### 🐍 Python (продвинутый уровень)
- **Что делал:** абстрактные классы, аннотации типов (все проекты), декораторы (Bank Fixtures), генераторы (HW Bank), тестирование (pytest + coverage).

### 🌐 Django & Django REST Framework
- **Проекты:** LMS, Atomic Habit Tracker, Mindself Psy Manufactory, SkyStore.
- **Что делал:** CRUD, JWT-аутентификация, разграничение прав (модераторы, владельцы), пагинация, фильтрация, кастомные валидаторы, документация через drf-spectacular.
- **Где пригодится:** быстрая разработка API для мобильных приложений, админок, CRM.

### 🗄️ PostgreSQL & SQLite
- **Проекты:** HH Parser & Loader (загрузка 10+ компаний, сложные запросы), LMS (модели курсов, подписок), Mindself (пользователи, тесты, консультации).
- **Что делал:** проектирование схем, индексация, написание сложных выборок (DBManager), миграции.
- **Где пригодится:** оптимизация запросов, работа с большими объёмами данных, поддержка целостности.

### 🔄 Celery + Redis
- **Проект:** Atomic Habit Tracker.
- **Что делал:** отложенные задачи (напоминания в Telegram), периодические задачи через django-celery-beat, настройка брокера.
- **Где пригодится:** асинхронная обработка уведомлений, генерация отчётов, фоновые задачи без блокировки основного потока.

### 🐳 Docker & Docker Compose
- **Проекты:** LMS, Atomic Habit Tracker, Mindself.
- **Что делал:** контейнеризация приложений, связка Django + PostgreSQL + Redis + Celery в одном compose-файле.
- **Где пригодится:** воспроизводимое окружение, упрощение деплоя, изоляция сервисов.

### ☁️ CI/CD (GitHub Actions)
- **Проект:** Atomic Habit Tracker.
- **Что делал:** автоматический прогон тестов, линтеров (flake8, black, isort, mypy) при push.
- **Где пригодится:** контроль качества кода перед слиянием, автоматическая проверка PR.

- **Проекты:** HW Bank, Bank Fixtures, парсеры hh.ru.

### 🤖 Telegram-боты
